## Isinya Apaan Aja?

Repository ini berisi kumpulan materi, contoh program, dan tugas praktikum untuk mata kuliah **Dasar-Dasar Pemrograman** menggunakan bahasa C. Modul ini dirancang untuk membantu mahasiswa memahami konsep fundamental pemrograman secara hands-on.

## Cara Menggunakan

### **Compile & Run**

**Menggunakan Dev-C++:**
1. Open file `.c` → Compile (F9) → Run (F10)
2. Atau langsung: Compile & Run (F11)

**Menggunakan GCC (Command Line):**
```bash
gcc nama_file.c -o output
./output
```

**Menggunakan VS Code:**
1. Install extension "C/C++" dari Microsoft
2. Buka file → Run → Start Debugging (F5)

**Menggunakan CLion:**
1. Create New Project → C Executable
2. Tulis code → Run (Shift+F10)
